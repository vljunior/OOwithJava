A forma mais simples é usar o Spring Initializr:

https://start.spring.io

Selecione:

Project: Maven (ou Gradle)

Language: Java

Spring Boot: 3.x (atual)

Dependencies:

Spring Web

Spring Data JPA

Driver do banco (ex.: MySQL ou H2)

Isso gera um esqueleto pronto.

Estrutura

src/main/java/com/exemplo/escola/
 ├─ EscolaApplication.java
 ├─ model/
 │   └─ Aluno.java
 ├─ repository/
 │   └─ AlunoRepository.java
 ├─ controller/
 │   └─ AlunoController.java
 └─ service/
     └─ AlunoService.java


Se usar MySQL:

spring.datasource.url=jdbc:mysql://localhost:3306/escola
spring.datasource.username=root
spring.datasource.password=1234
spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true


Pode usar H2 (em memória):

spring.datasource.url=jdbc:h2:mem:escola
spring.datasource.driver-class-name=org.h2.Driver
spring.jpa.hibernate.ddl-auto=update
spring.h2.console.enabled=true
